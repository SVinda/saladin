Listas de archivos

La lista de carpetas y archivos del directorio actual es la parte principal de cada panel. La lista se actualiza automáticamente cuando se producen cambios en el directorio; también puede seleccionar Visualizar > Actualizar (Ctrl+R) para actualizar la lista.

Haga doble clic en un elemento para abrir el archivo, la carpeta o el archivo ZIP. Es lo mismo que pulsar Intro o la tecla de dirección Derecha sobre el elemento actual. Puede utilizar el elemento especial con nombre .. para acceder a la carpeta anterior, a no ser que la carpeta actual ya sea el directorio raíz de una unidad o el Escritorio. También puede utilizar el comando Directorio principal, la tecla Retroceso o la tecla de dirección Derecha para ir al directorio principal.

La primera columna de la lista muestra el nombre del archivo con su extensión. La segunda columna muestra el tamaño del archivo en bytes, <DIR> para las carpetas y <LNK> para vínculos físicos y uniones. El tamaño total de una carpeta puede calcularse y mostrarse si presiona la tecla Espacio, o si selecciona una o más carpetas y elige Herramientas > Mostrar tamaño (Ctrl+L). La tercera columna muestra la fecha y la hora en la que se modificó por última vez el archivo.

La última columna muestra los siguientes atributos: d de carpeta, r de solo lectura, a de archivo, h de oculto, s de sistema, c de comprimido y e de encriptado. Se muestra una letra cuando se ha establecido un atributo, si no es el caso, se muestra un guion. Además, los archivos ocultos se muestran en color gris y los comprimidos, en color azul oscuro. Los archivos ocultos o del sistema no se muestran por defecto; pueden aparecer si los activa con la opción Visualizar > Archivos ocultos o Ctrl+H.

Puede organizar la lista por cualquier columna al hacer clic en el encabezado de la columna. Tenga en cuenta que las carpetas se muestran siempre antes que los archivos, aunque se pueden ordenar por tamaño, fecha o atributos. También puede reorganizar las columnas o cambiar su tamaño si arrastra los encabezados o los separadores. El diseño de las columnas se cambia automáticamente en los dos paneles.

Selección

El elemento actual del panel activo se marca con un cursor (un marco fino y azul). Los archivos seleccionados en los dos paneles se muestran con la fuente en color rojo. La mayoría de las operaciones se realizan en todos los elementos seleccionados en el panel activo o en el elemento actual si no se ha seleccionado ninguno.

Puede utilizar las teclas de dirección Arriba y Abajo, las teclas Retroceder página, Avanzar página, Inicio y Fin para mover el cursor entre la lista. También puede hacer clic en un elemento con el botón izquierdo del ratón para mover el cursor hasta la posición elegida y quitar la selección de todos los elementos seleccionados previamente. Teclee la primera letra (o las primeras letras) para llevar el cursor hasta el siguiente elemento que empiece con esas letras.

Puede seleccionar o quitar de la selección un intervalo de elementos al pulsar la tecla Shift y al hacer clic con el botón izquierdo del ratón o cambiando el elemento actual con el teclado. También puede alternar la selección con elementos individuales al pulsar la tecla Ctrl y haciendo clic en el elemento con el botón izquierdo del ratón. La tecla Insert alterna la selección del elemento actual y mueve el cursor abajo. La tecla Espacio alterna la selección del elemento actual sin mover el cursor, y también calcula el tamaño total del directorio seleccionado.

Para seleccionar varios archivos según su nombre, utilice el comando Seleccionar > Seleccionar máscara (Ctrl+=). Puede escribir el patrón mediante caracteres comodín, por ejemplo, *.txt. Puede escribir varios patrones si los separa con un punto y coma, por ejemplo, *.txt;*.ini. El comando Seleccionar > Anular selección de máscara (Ctrl+-) funciona de la misma manera, pero anula la selección de los archivos que coincidan con la máscara. Los comandos Seleccionar > Seleccionar todo (Ctrl+A), Seleccionar > Anular selección (Ctrl+Shift+A) y Seleccionar > Invertir selección (Ctrl+-I) también pueden utilizarse para modificar la selección en el panel activo.

Menús contextuales

Pulse el botón derecho del ratón en un elemento para mostrar el menú contextual para ese elemento. Seleccione varios elementos y haga clic con el botón derecho del ratón en uno de los elementos seleccionados para mostrar el menú contextual para esos elementos. Haga clic fuera de cualquier elemento o en .. para mostrar el menú contextual para el directorio actual. Al hacer clic con el botón derecho en la barra de discos, se mostrará el menú contextual del disco.

Cambio de nombre

Haga clic en el nombre del archivo actual con el botón izquierdo para cambiarle el nombre; el comando Funciones > Cambiar nombre o la tecla F2 también se pueden utilizar con este propósito. Pulse Intro para confirmar el nombre nuevo o Esc para cancelar el cambio de nombre.

Arrastrar y soltar

Puede arrastrar uno o varios elementos seleccionados al mantener pulsado el botón izquierdo o derecho del ratón. Al arrastrar elementos sobre una carpeta o un archivo ZIP, se marca con un marco fino y negro que indica que los elementos pueden copiarse o moverse a esa ruta. Si arrastra un elemento sobre un archivo normal o sobre una zona vacía de la lista, se arrastrará al directorio actual. Si arrastra archivos al elemento .., se enviarán al directorio anterior. Si arrastra archivos sobre un botón de la barra de unidades, se arrastrarán al directorio raíz del disco.

Mantén presionado Ctrl para copiar archivos, Shift para mover o Alt para crear un vínculo simbólico. La operación por defecto depende del contexto. Cuando esté arrastrando archivos con el botón derecho del ratón, se muestra un menú contextual para que elija la operación que prefiera.

Se pueden arrastrar archivos de Saladin a otros programas y viceversa. Las operaciones normales del Portapapeles (cortar, copiar y pegar) pueden usarse para copiar y mover archivos entre directorios dentro de Saladin y desde o hacia otros programas.
