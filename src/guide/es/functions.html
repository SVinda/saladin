Funciones básicas

Visualizar

El comando Visualizar (F3) abre el archivo actual (el que está bajo el cursor) con el visor interno o externo. Por defecto se utiliza el visor interno. También puede utilizar otra aplicación como visor y seleccionarla en la Configuración de Saladin.

El comando Ver archivos seleccionados (Shift+F3) hace que sea posible visualizar rápidamente todos los archivos actualmente seleccionados. Esta acción solo está disponible cuando se utiliza el visor interno. En un primer momento, se abre el primer archivo seleccionado; se puede cambiar entre archivos seleccionados mediante los comandos Anterior (P o Alt+LEFT) y Siguiente (N o Alt+RIGHT) del visor.

Editar

El comando Editar (F4) abre el archivo actual para su edición en un editor de texto. Por defecto se utiliza el Bloc de notas, pero puede instalar un editor más potente como el Notepad++ y seleccionarlo en la Configuración de Saladin. Puede utilizar el comando Editar > Crear archivo (Shift+F4) para crear un nuevo archivo vacío en la carpeta actual y abrirlo con el editor de texto.

Copiar y mover

Se pueden copiar los archivos y las carpetas seleccionados de la siguiente manera: Copiar (F5), o moverlos mediante Mover (F6). Puede clonar los archivos seleccionados en el directorio de origen mediante Copiar > Clonar en el directorio de origen (Shift+F5). Puede cambiar el nombre de varios archivos a la vez mediante Mover > Renombrar en el directorio de origen (Shift+F6). También se puede clonar o cambiar el nombre a varios archivos cuando los directorios de origen y de destino son el mismo.

Cuando se copia o se mueve un solo elemento, su nombre se puede cambiar directamente en el cuadro de diálogo para confirmar la acción. Cuando se copian o se mueven varios elementos, puede abrir el panel de cambio de nombre múltiple al hacer clic en el botón Cambiar nombre (se abre automáticamente cuando se clona o se cambia de nombre). En algunos casos, no se puede cambiar el nombre porque Windows no lo permite.

En el panel de cambio de nombre, se pueden utilizar máscaras especiales en los campos Nombre y Extensión para crear el nuevo nombre de los elementos seleccionados. Además, cualquier parte del nombre se puede reemplazar mediante las opciones de Buscar y Reemplazar. Si está activada la opción de Expresiones regulares, la cadena de texto que se vaya a buscar puede contener expresiones regulares. El reemplazo se realiza después de la creación del nombre a partir de las máscaras. Por último, los caracteres del nombre resultante se pueden convertir a mayúsculas o minúsculas con la opción Mayúsculas y minúsculas. La lista de la parte final del panel de cambio de nombre múltiple muestra el nombre original y el nuevo de cada archivo.

Cuando la acción de mover archivos requiera tiempo, se mostrará un cuadro de diálogo con el progreso. Se muestran mensajes de aviso cuando se intenta sobrescribir un archivo existente. Cuando la opción de Sobrescribir sin preguntar está activada, los archivos seleccionados se reemplazan sin previo aviso.

Crear carpeta

Puede crear una nueva carpeta con el nombre que desee en el directorio actual mediante Crear carpeta (F7).

Eliminar

Puede utilizar el comando Eliminar o las teclas F8 o Suprimir para eliminar los archivos seleccionados. Los archivos y las carpetas se envían a la Papelera de reciclaje si está habilitada en ese disco. El comando Eliminar > Eliminar permanentemente o las teclas Shift+F8 o Shift+Supr eliminan los archivos sin enviarlos a la Papelera de reciclaje. En ambos casos se muestra un aviso para confirmar la acción.

Abrir consola

La última opción básica es Abrir consola (F9), que sirve para abrir el Símbolo del sistema en el directorio actual. Por defecto, se utiliza el programa cmd.exe pero puede seleccionar otra consola. Saladin no tiene ningún intérprete de comandos incorporado.
