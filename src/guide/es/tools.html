Herramientas

Ficheros ZIP

Saladin soporta archivos comprimidos en ZIP debido a la funcionalidad estándar del Explorador de Windows. Puede abrir un archivo ZIP haciendo doble clic o pulsando Intro, igual que una carpeta normal. Puede extraer o añadir archivos de ficheros ZIP mediante la operación de arrastrar y soltar. También puede crear carpetas y renombrar y borrar archivos dentro de un fichero ZIP. Para añadir archivos a un fichero ZIP, seleccione los archivos que desee comprimir, seleccione el comando Herramientas > Comprimir (Ctrl+P) y escriba el nombre del fichero en el directorio de destino. Para trabajar con otros tipos de archivos comprimidos instale una herramienta externa que se integre con Windows Shell como, por ejemplo, 7-Zip.

Conexiones FTP

Saladin también tiene un soporte básico para el protocolo FTP a través de la funcionalidad del Explorador de Windows. Para conectarse a un servidor FTP, seleccione el comando Herramientas > Abrir Conectar a FTP (Ctrl+F) y escriba el nombre del servidor o dirección IP, opcionalmente puede cambiar el número de puerto y la ruta de inicio. También puede escribir el usuario y la contraseña si el servidor las pide. La URL del sitio se puede escribir directamente en la barra de direcciones. Las operaciones normales como copiar, mover, cambiar de nombre y eliminar se pueden realizar en conexiones FTP.

Buscar y explorar

Para buscar archivos en el directorio actual y en sus subcarpetas, seleccione Herramientas > Buscar (Ctrl+S). Introduzca el patrón mediante caracteres comodín, por ejemplo *.txt. Puede introducir varios patrones si los separa con un punto y coma, por ejemplo, *.txt;*.ini. También puede buscar archivos que contengan un texto concreto. Active la opción Texto e introduzca el texto que quiera buscar. Active Diferenciar mayúsculas y minúsculas si desea realizar una búsqueda que diferencie las mayúsculas de las minúsculas. Cuando busque un texto, los archivos binarios se excluirán automáticamente.

La lista de resultados de búsqueda es similar a la lista de la ventana principal, excepto que se muestra la ruta del directorio actual. La lista también muestra el tamaño del archivo en bytes, la fecha y la hora en la que se modificó por última vez, y los atributos del archivo. Puede organizar la lista por cualquier columna al hacer clic en el encabezado de la columna. Pulse el botón derecho del ratón en un elemento para mostrar el menú contextual para ese elemento.

Utilice el comando Visualizar (F3) para abrir el archivo actual con el visor en los resultados de búsqueda. Utilice el comando Editar (F4) para abrir el archivo actual para su edición en un editor de texto. El comando Ir a archivo (Ctrl+G) cerrará los resultados de búsqueda y resaltará el archivo seleccionado en el panel activo. El comando Copiar los nombres de archivo (Ctrl+Shift+C) se puede usar para copiar las rutas de acceso relativas de todos los archivos al Portapapeles.

También puede abrir el archivo actual en el Explorador de Windows con Herramientas > Explorar (Ctrl+Shift+E).

Comparar

Puede comparar dos archivos con una herramienta externa como WinMerge o la aplicación de TortoiseMerge TortoiseSVN. Para configurar la herramienta abra la Configuración de Saladin y seleccione la ruta del ejecutable del programa. Puede seleccionar dos archivos en el panel activo o uno en el activo y el otro en el panel de destino, entonces seleccione el comando Herramientas > Comparar (Ctrl+M) para ejecutar la herramienta de comparación.

Utilice el comando Herramientas > Comparar > Comparar directorios (Ctrl+Shift+M) para comparar los archivos en el directorio actual y en el de destino. Este comando selecciona todos los archivos y carpetas del directorio actual que no están en el directorio de destino, y los archivos que están más actualizados que en el directorio de destino. Es útil a la hora de sincronizar el contenido de dos directorios.

Marcadores

Los marcadores se pueden utilizar para recordar los directorios locales y las direcciones FTP que utilice frecuentemente. Haga clic en el icono Marcadores y seleccione Añadir marcador (Ctrl+Shift+D) para guardar la ruta actual como marcador con el nombre que elija. Cuando guarde un FTP como marcador, puede elegir guardar la contraseña.

Para abrir un marcador, haga clic en el icono Marcadores o pulse Ctrl+D y seleccione el marcador de la lista. Cuando se activa un marcador FTP, la ventana de Abrir FTP aparece para que pueda escribir su contraseña o para cambiar detalles de la dirección sin modificar el marcador original. Puede eliminar y cambiar el nombre a los marcadores existentes accediendo a Editar marcadores del menú de Marcadores. Los marcadores se comparten entre los dos paneles y se guardan automáticamente cuando se cierra Saladin.

Historial

Cada panel guarda las carpetas que se han visitado anteriormente. Puede navegar a direcciones anteriores o posteriores con las combinaciones de teclas Alt+Derecha y Alt+Izquierda, respectivamente; o utilizando los botones adicionales del ratón si el suyo los tiene. También puede seleccionar una de las direcciones de la lista al hacer clic en el icono Historial o pulsando Ctrl+G.
