Guía rápida de Saladin

La ventana principal de Saladin contiene una barra de herramientas para acceder a las funciones y a las herramientas más usadas. Se puede ocultar la barra de herramientas con el botón Contraer barra de herramientas en la esquina superior derecha de la ventana. Cuando la barra de herramientas está contraída, se muestra una barra de menú que contiene todas las funciones y herramientas. Para volver a mostrar la barra de herramientas, utilice el botón Expandir barra de herramientas.

Por debajo de la barra de herramientas, hay dos paneles para trabajar con los directorios. Cada panel consta de barra de discos, barra de ubicaciones, lista de archivos y barra de estado. Uno de los paneles está activo y actúa como el panel de origen para las acciones de copiar y mover; el otro panel es el de destino. Puede hacer clic en cualquier parte del panel para activarlo o pulsar Tab para intercambiar el panel activo. Los paneles de origen y destino se diferencian por el color de la barra de direcciones.

La barra de discos muestra un botón que representa a cada disco duro disponible o a otros discos conectados al ordenador. Puede hacer clic en el botón del disco para abrir el directorio raíz del disco en el panel de debajo. También puede pulsar Alt+F1 o Alt+F2 para seleccionar el disco desde una lista desplegable en el panel izquierdo o en el derecho, respectivamente. Haga clic en la barra de discos y seleccione Actualizar unidades, en caso de que la barra no se actualice automáticamente al añadir o expulsar una unidad.

La barra de direcciones muestra la ruta del directorio actual o el nombre, en caso de que no tenga una ruta física. Puede escribir la ruta del directorio en la barra de direcciones y presionar Intro para abrirla. Haga clic en la barra de direcciones o pulse Ctrl+O para activarla. También puede seleccionar Abrir (Ctrl+Shift+O) para acceder a un directorio.

La lista de archivos muestra el contenido del directorio actual.

La barra de estado muestra la cantidad y el tamaño de los elementos seleccionados comparados con todos los elementos del directorio actual. También muestra el nombre, la capacidad y la cantidad de espacio disponible del disco actual.
