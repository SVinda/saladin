Tools

ZIP Archives

Saladin can handle ZIP archives using standard functionality of Windows Explorer. You can open a ZIP archive like a regular directory using double click or Enter. You can extract files from ZIP archives and add files to existing archives using standard copy operation or drag and drop. You can also create folders and rename and delete files and folders in a ZIP archive. To add files to a ZIP archive, select files to pack, select the Pack command (Ctrl+P) and enter the name of the achive in the target directory. For working with other archive types install an external tool which integrates with Windows Shell, for example 7-Zip.

FTP Connections

Saladin also has basic support for the FTP protocol through Windows Explorer functionality. To connect to an FTP server, select Open > Connect To FTP command (Ctrl+F) and enter the host name or IP address and optionally change the port number and the initial path. You can also enter user name and password if the server requires authentication. The URL of the site can also be entered directly in the location bar. Standard file operations like copying, moving, renaming and deleting are supported for FTP connections.

Search

To search for files in the current directory and its subdirectories, select Search (Ctrl+S). Enter the pattern using wildcards, for example *.txt. You can also enter multiple patterns by separating them with a semicolon, for example *.txt;*.ini. You can also search for files containing the given text. Enable the Text option and enter the text you want to search for. Enable Match case if you want to perform a case-sensitive search. When searching for a text, binary files will be automatically excluded.

The list of search results is similar to the lists in the main window, except that the path relative to the current directory is displayed. The list also displays the size of the file in bytes, the date and time when the file was last modified, and the attributes of the file. You can sort the list by any column by clicking on the column header. You can also press the right mouse button on an item to show the contextual menu for that item and double click to open it in the default application.

Use the View command (F3) to open the current file in the search results using the viewer. The View All command (Shift+F3) makes it possible to quickly view all search results in the internal viewer. Initially the first matching file is opened; it is possible to switch between matching files using the Previous command (P or Alt+LEFT) and the Next command (N or Alt+RIGHT) in the viewer.

Use the Edit command (F4) to open the current file for editing in a text editor. The Go To File command (Ctrl+G) will close the search results and highlight the selected file in the active panel. The Copy File Names command (Ctrl+Shift+C) can be used to copy the relative paths of all files to the Clipboard.

Explore

You can open the current directory in Windows Explorer using Explore (Ctrl+Shift+E).

Compare

You can compare two files using an external tool such as WinMerge or the TortoiseMerge utility from TortoiseSVN. To configure the tool open Saladin Settings and select the path to the tool's executable file. You can select two files in the active panel or one in the active and the other one in the target pane and then select the Compare command (Ctrl+M) to run the compare tool.

Use the Compare > Folder Compare command (Ctrl+Shift+M) to compare items in the current and the target directories. This command selects all files and folders from the current directory which are not present in the target directory and files that are newer than those in the target directory. This is useful for synchronizing the contents of two directories.

Bookmarks

Bookmark can be used to remember frequently used local directories and FTP sites. Click on the Bookmarks icon and select Add Bookmark (Ctrl+Shift+D) to save the current location as a bookmark with the given name. When an FTP site is added as a bookmark, you can choose to store the password.

To open a bookmark, click on the Bookmarks icon or press Ctrl+D and select the bookmark from the list. When an FTP bookmark is activated, the Open FTP Site window is opened allowing to enter password or change the details of the FTP site without modifying the original bookmark. You can delete and rename existing bookmarks by selecting the Edit Bookmarks command from the Bookmarks menu. Bookmarks are shared by both panels and automatically remembered when Saladin is closed.

History

Each panel remembers last visited directories. You can navigate to previous or next location using Alt+Right and Alt+Left keys, respectively, or using the additional mouse buttons if your mouse supports ones. You can also select one of the locations from the list by clicking on the History icon or pressing Ctrl+G.
