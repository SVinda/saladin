Internal Viewer

The internal viewer can be opened using the View command (F3). All currently selected files can be quickly viewed using the View Selected Files command (Shift+F3). When multiple files are opened, it is possible to switch between selected files using the Previous command (P or Alt+LEFT) and the Next command (N or Alt+RIGHT) in the viewer.

The viewer automatically recognizes the format of the file and displays it in one of the following modes:

  • Text
  • Binary
  • Image

You can switch between those modes using the 1, 2 and 3 keys or the toolbar buttons. You can also reload the file to reflect the latest changes using the Reload command (Ctrl+R).

In the text mode the file is displayed using automatically detected encoding, either UTF-8, ANSI (the default encoding configured in Windows), Unicode or Unicode BE (big endian). You can select an appropriate encoding from the Encoding menu if it wasn't detected correctly. It is possible to select and copy a fragment of text to the Clipboard and to enable or disable wrapping long lines to the edge of the window using the Word Wrap option (W). You can also use the Find command (Ctrl+F) to quickly search for any text in the file using the search bar. You can also go to the specified line using the Go To Line command (Ctrl+G) and toggle displaying line numbers on or off using the Line Numbers command (L). To open the currently viewed file in the text editor, use the Edit File command (F4).

In the binary mode the file is displayed using a fixed number of 80 bytes per row, with non-printable characters replaced with a dot. You can enable the Hexadecimal option (H) to diplay 16 bytes per row, along with the offset and hexadecimal values for each byte. It is also possible to select and copy a fragment of text to the Clipboard in this mode.

In the image mode a PNG, JPEG, GIF, BMP, TIFF, ICO or SVG image can be displayed. It is possible to copy the entire image to the Clipboard. You can enable the Zoom To Fit option (F) to automatically scale the image to the window if it's too large. You can also zoom the image in and out, rotate it by 90 degrees in both directions, and change the background color of the image between black and white. To display information about the image, such as the format, size and number of colors, use the Information command (I). You can also switch the viewer to full screen mode using the Full Screen command (F11).

You can change the font and text size used in text and binary views in Saladin Settings. The font used for the binary view must be a fixed width font. Fonts with ligatures, such as Fira Code, are supported if the font is installed in your system.

Summary of keyboard shortcuts in the internal viewer:

Key Function
1 Switch to the text mode.
2 Switch to the binary mode.
3 Switch to the image mode.
A Select ANSI encoding (text mode only).
B Switch the background color between black and white.
F Fit the image to the window (image mode only).
H Enable or disable hexadecimal display (binary mode only).
I Display information about the image (image mode only).
L Toggle line numbers on and off (text mode only).
L Rotate the image counter-clockwise (image mode only).
N View the next selected file.
O Select OEM encoding (text mode only).
P View the previous selected file.
R Rotate the image clockwise (image mode only).
U Select UTF-8 encoding (text mode only).
W Enable or disable word wrapping (text mode only).
F3 Find the next occurrence of the last searched text (text mode only).
Shift+F3 Find the previous occurrence of the last searched text (text mode only).
F4 Open the current file in the text editor (text mode only).
F11 Toggle full screen mode.
ESC Close the viewer window.
Alt+LEFT View the previous selected file.
Alt+RIGHT View the next selected file.
Ctrl+- Zoom out the image (image mode only).
Ctrl+= Zoom in the image (image mode only).
Ctrl+0 Display the image in the original size (image mode only).
Ctrl+A Select all text (text and binary mode).
Ctrl+C Copy selected text or current image to the Clipboard.
Ctrl+F Find specified text (text mode only).
Ctrl+G Go to specified line (text mode only).
