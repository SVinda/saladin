Keyboard Shortcuts

Summary of keyboard shortcuts in Saladin:

Key Function
F1 Display information about Saladin.
F2 Rename the current item in-line.
F3 View the current item using the viewer tool.
F4 Edit the current item using the text editor tool.
Shift+F4 Create a new, empty text file and edit it using the text editor tool.
F5 Copy selected items to the target directory.
Shift+F5 Clone selected items within the source directory.
F6 Move selected items to the target directory.
Shift+F6 Rename selected items within the source directory.
F7 Create a new folder.
F8 Move selected items to the Recycle Bin.
Shift+F8 Permanently delete selected items.
F9 Open Command Prompt in the current directory.
Alt+F1 Select drive in the left panel.
Alt+F2 Select drive in the right panel.
TAB Switch the active panel.
LEFT Go to parent directory.
RIGHT Enter current directory or ZIP archive.
Ctrl+LEFT Open parent directory in the other panel.
Ctrl+RIGHT Open current directory or ZIP archive in the other panel.
Alt+LEFT Go to previous directory in history.
Alt+RIGHT Go to next directory in history.
ENTER Open current file or enter current directory or ZIP archive.
Alt+ENTER Show propertis of selected items.
SPACE Toggle selection of current item and calculate the size of the directory.
INSERT Toggle selection of current item and move to next item.
DELETE Move selected items to the Recycle Bin.
Shift+DELETE Permanently delete selected items.
BACKSPACE Go to parent directory.
Ctrl+\ Go to the root directory.
Ctrl+= Select items matching specified pattern.
Ctrl+- Unselect items matching specified pattern.
Ctrl+A Select all items.
Ctrl+Shift+A Unselect all items.
Ctrl+C Copy selected items to the Clipboard.
Ctrl+Shift+C Copy the names of selected items.
Ctrl+D Open the list of bookmarks.
Ctrl+Shift+D Add current directory to the list of bookmarks.
Ctrl+E Open the same directory in the other panel.
Ctrl+Shift+E Open Windows Explorer in the current directory.
Ctrl+F Open an FTP site.
Ctrl+G Open the history of directories.
Ctrl+H Enable or disable displaying hidden files.
Ctrl+I Invert selection.
Ctrl+L Calculate the size of selected items.
Ctrl+M Compare two files using external compare tool.
Ctrl+Shift+M Compare the contents of the source and target directories.
Ctrl+O Open a directory using the location bar.
Ctrl+Shift+O Open a directory using the directory browser.
Ctrl+P Pack selected files to a new ZIP archive in the target directory.
Ctrl+R Refresh the current directory.
Ctrl+S Search for files in the current directory.
Ctrl+U Swap the source and target directories.
Ctrl+V Paste items from the Clipboard to the current directory.
Ctrl+X Cut selected items to the Clipboard.
