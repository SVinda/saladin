Basic Functions

View

The Functions > View command (F3) opens the current file (the one under the cursor) using an external viewer application. By default no viewer is available. You can install a tool such as the free Universal Viewer and select the path to Viewer.exe in Saladin Settings. In the next version Saladin will use its own internal viewer by default.

Edit

The Functions > Edit command (F4) opens the current file for editing in a text editor. By default the standard Notepad application is used but you can install a more powerful editoTr such as Notepad++ and select it in the Saladin Settings. You can use the Functions > Edit > Create File command (Shift+F4) to create a new empty file with the given name in the current directory and open it in the editor.

Copy & Move

Selected files and directories can be copied using Functions > Copy (F5) and moved using Functions > Move (F6). Use Functions > Copy > Clone In Source Directory (Shift+F5) to clone the selected items in the source directory. Use Functions > Move > Rename In Source Directory (Shift+F6) to rename multiple items at once. Cloning or multi-renaming also occurs when the source and target directories are the same.

When copying or moving a single item, its name can be changed directly in the confirmation dialog. When copying or moving multiple items, the multi-rename panel can be opened by clicking the Rename button (it is automatically opened when cloning or renaming). In some cases renaming is not allowed when Windows does not support it.

In the multi-rename panel special masks can be entered in the Name and Extension fields to build the new name of selected items. Also any part of the name can be replaced using Search and Replace options. If the Regular expressions option is enabled, the search string can contain a regular expression. Replacement is done after building the name from the masks. Finally the characters in the resulting names can be converted to upper/lower case using the Case option. The list in the bottom part of the multi-rename panel displays the original name and the new name of each item.

For long operations a standard progress dialog is displayed while transfering the files. Warning messages are displayed when the operation attempts to overwrite an existing file. When the Overwrite without prompt option is selected files are replaced without additional warnings.

Create Folder

Use Functions > Create Folder (F7) to create a new subdirectory with the given name in the current directory.

Delete

Use the Functions > Delete command or the F8 or Delete key to delete selected items. Files and directories are moved to the Recycle Bin if it is enabled on the given drive. The Functions > Delete > Delete Permanently command or the Shift+F8 or Shift+Delete key deletes the items without using the Recycle Bin. In both cases an additional warning is displayed to confirm the operation.

Open Console

The last basic function is Functions > Open Console (F9) which opens the Command Prompt in the current directory. By default the standard cmd.exe application is used but you can select a different console. Saladin does not have an integrated command line interpreter.
